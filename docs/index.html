One task system for
you and your agents.

A lightweight, agents-first task tracker. Work from the terminal or web while your coding agents manage the same tasks through structured MCP tools. One self-hostable ~13 MB Go binary, backed by SQLite.

Humans and agents, one source of truth

Capture work in the terminal, review it in a browser, and let coding agents handle task CRUD through MCP. Every surface updates the same tasks.

CLIFast capture, scripts, JSON / NDJSON for piping into jq.
TUIBubble Tea today/inbox views. vim-style keys, runs over SSH.
Web appInstallable on mobile and desktop. Today, focus, trends, projects, and settings. No build step.
REST APIStable /api/v1 surface, session cookie or X-API-Key.
MCP interfaceStructured tools for Claude, Codex, Cursor, Cline, and friends. Built in, with no extra service to deploy.
WebSocketLive updates on /api/v1/ws for the TUI and web UI.

Agents-first without the infrastructure tax

No Electron, Node runtime, or external database. Drop one static binary on a server and give people and agents a shared place to manage work.

M

Multi-tenant by design

Every row is scoped by tenant_id at the storage layer. Cross-tenant access is structurally impossible, not just policy.

T

Timers + Pomodoro

Start/stop a work timer on any task. Run a 25-minute pomodoro. Get a per-day worklog broken down by task.

R

Recurring tasks

iCalendar RRULE. Mark a weekly standup done and the next occurrence spawns automatically.

!

Reminders

Schedule a fire-at timestamp. Background ticker publishes to the WebSocket hub and fires a best-effort OS notification.

A

Agent-ready in one command

ttl agents install installs the skill and registers structured MCP tools for supported coding agents without copying your API key into their configs.

$

One file, one process

SQLite + WAL handles hundreds of writes per second. Back up with cp. No services to register.

See it in action

The installable web app is vanilla JS with no build step or framework. Review today's work, run a Pomodoro, and see 14-day productivity trends from any device.

Today keeps due work, tasks completed today, focus controls, daily metrics, and lightweight 14-day trends together in one view.
