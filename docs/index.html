The task tracker that
stays out of your way.

A single static Go binary (~13 MB) that ships a CLI, TUI, web UI, REST API, WebSocket live updates, and an MCP server for AI agents. SQLite + WAL, no CGO, row-level multi-tenancy.

Six surfaces, one binary

Use whatever fits the moment: terminal at your desk, browser on your phone, an MCP tool call from your agent.

CLIFast capture, scripts, JSON / NDJSON for piping into jq.
TUIBubble Tea today/inbox views. vim-style keys, runs over SSH.
Web UIEmbedded SPA. Today, inbox, projects, settings. No build step.
REST APIStable /api/v1 surface, session cookie or X-API-Key.
MCP server10 tools over stdio for Claude, Cursor, Cline, and friends.
WebSocketLive updates on /api/v1/ws for the TUI and web UI.

Built for speed and keyboard

No Electron. No Angular. No Docker required. Just a single static binary you can drop on a server and forget about.

M

Multi-tenant by design

Every row is scoped by tenant_id at the storage layer. Cross-tenant access is structurally impossible, not just policy.

T

Timers + Pomodoro

Start/stop a work timer on any task. Run a 25-minute pomodoro. Get a per-day worklog broken down by task.

R

Recurring tasks

iCalendar RRULE. Mark a weekly standup done and the next occurrence spawns automatically.

!

Reminders

Schedule a fire-at timestamp. Background ticker publishes to the WebSocket hub and fires a best-effort OS notification.

A

AI-native

Stable REST API, NDJSON streaming, MCP server. Programmable from any language, native to every major agent.

$

One file, one process

SQLite + WAL handles hundreds of writes per second. Back up with cp. No services to register.

See it in action

Same UI as ttl on the wire. The web client is ~12 KB of vanilla JS — no build step, no framework.

Inbox shows all root-level open tasks. Today filters to what's due, plus an active timer banner and per-day worklog from /api/v1/worklog/today.
