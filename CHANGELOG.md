# Changelog

## v1.1.0 — 2026-07-13

- Made Today the safe default for new tasks without a due date and added a
  completed-today section based on completion time.
- Added web Pomodoro controls, clearer recurrence presets, task-dialog close
  controls, daily productivity metrics, and lightweight 14-day trends.
- Made the web UI an installable PWA for mobile and desktop while keeping task,
  authentication, timer, and analytics data out of the offline cache.
- Improved agents-first onboarding and product copy across the site and docs.

## v1.0.0 — 2026-07-13

- Recoverable task trash, restore, permanent purge, manual ordering, and smart views.
- Complete task updates, recurrence presets, reminders, subtasks, and bounded Pomodoros.
- Scoped and expiring API keys with rename/rotation/revocation, team invites, and member administration.
- Signed reminder webhooks and hardened HTTP defaults.
- Expanded CLI, MCP, web, API, migration, and release verification.
